# Asean Youth Forum (AYO) Project API

Build json server to facilitate the development process of the project group.

## Installation

Install with npm or yarn

```bash
  npm install
```

```bash
  yarn
```

Run project

```bash
  npm run start
```

```bash
  yarn start
```

Development

```bash
  npm run dev
```

```bash
  yarn dev
```

## Tech Stack

Json Server, git, github, vercel
